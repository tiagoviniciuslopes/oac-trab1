Linguagem: Java

O projeto foi dividido em algumas classes:
	- Instrucao: classe que representa instrucões e que faz a decodificação das instruções
	- Main: classe principal utilizada para executar o programa
	- Memoria: classe que representa a memoria
	- PC: classe que representa o program counter (PC)
	- Processador: classe que representa a unidade de controle e toma as decisões que os MUX tomariam
	- Registrador: classe que representa os 32 registradores
	- ULA: classe que representa a ULA/ALU

Utilização:
	1. Execute o comando "java Main"
	2. O programa irá exibir uma mensagem e vai aguardar a entrada de dados
	3. Entre com a instrução binária (o programa executa a instrução assim que a tecla enter for apertada)
	4. O programa mostra na tela os registradores e a memória utilizados durante a execução de todas as instruções
